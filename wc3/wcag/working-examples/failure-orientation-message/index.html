Showing message asking to reorient device

The objective of this Failure is to describe the situation where users are unable to rotate their device because it is mounted in a fixed orientation (e.g., on the arm of a power wheelchair).

If web content is designed for portrait only and when landscape is detected, a message appears asking to reorient the device, these users will be unable to rotate and operate the content.
