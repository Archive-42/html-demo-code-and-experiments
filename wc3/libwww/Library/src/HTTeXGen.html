This module converts a structured stream from HTML into LaTeX format. The conversion is mostly a 1:1 translation, but as the LaTeX compiler is much more strict than a typical HTML converter some typographical constraints are put on the translation. Only text is translated for the moment.

This module is implemented by HTTeXGen.c, and it is a part of the W3C Sample Code Library.
