We use James Clark's expat XML parser which is very neat indeed. As the code doesn't come as a separate library, I included it in the libwww CVS code base where I compile is as one library: libexpat.a. See the external modules that libwww works with for details.

System dependencies

The wwwsys.h file includes system-specific include files and flags for I/O to network and disk. The only reason for this file is that the Internet world is more complicated than Posix and ANSI. 

#include "wwwsys.h"
