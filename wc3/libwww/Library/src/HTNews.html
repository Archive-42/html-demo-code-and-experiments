We define the max NNTP line as rather long as the result coming from a XOVER command needs it. If the line is longer then it is chopped, but we will almost have received the information we're looking for.

Setting Number of Articles to Show

You can set the number of news articles to be shown at a time. If you set the number to none (0) then all articles are shown at once. This is also the default behavior. 

extern BOOL HTNews_setMaxArticles (int new_max);
extern int HTNews_maxArticles (void);
