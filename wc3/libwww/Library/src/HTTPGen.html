The HTTP General Header stream generates the general headers of a HTTP request or response and writes it to the target which is normally either a HTTP response stream or a HTTP request stream.

This module is implemented by HTTPGen.c, and it is a part of the W3C Sample Code Library. 

#ifndef HTTPGEN_H
#define HTTPGEN_H

#include "HTStream.h"
#include "HTReq.h"

#ifdef __cplusplus
extern "C" { 
#endif

Streams Definition

This stream makes a general HTTP header before it goes into transparent mode. If endHeader is YES then we send an empty CRLF in order to end the header. 

extern HTStream * HTTPGen_new  (HTRequest * request, HTStream * target,
				BOOL endHeader, int version);

#ifdef __cplusplus
}
#endif

#endif  /* HTTPGEN_H */
