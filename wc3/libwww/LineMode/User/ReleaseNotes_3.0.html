Line Mode Browser 3.0 Release Notes

This release is the first public release of source code which is covered by the new CERN copyright statement. The statement is published in the file COPYRIGH (truncated to 8 bytes) in the distribution packet or in hypertext format.

As always, most of the added features in the Line Mode Browser are a result of changes to the Library of Common Code. Most of the code is reorganized in order to adapt the application to the multithreaded version of the Library. This means that the browser now has the following new features:

3.0, Marts 21 1995

Only very little has changed in this release. Most of the new features are in the Library.

3.0 Prerelease 2, December 2 1994

  • The browser is now much more stable and has the same amount of functionality as the previous version 2.15

3.0 Prerelease 1, November 26 1994

  • Interruptible I/O - hit 'z RETURN' and the current load terminates
  • You can at any time initiate a new load or any other command by simply start typing.
