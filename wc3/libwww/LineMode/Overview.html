WWW - The Libwww Line Mode Browser

The www is an example application that comes with libwww - the W3C Sample Code Library. It is a character based Web browser developed for use on terminals. It uses the libwww HTTP/1.1 implementation including a full HTTP/1.1 persistent cache, history lists etc.
