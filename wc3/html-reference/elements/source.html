The source element

media source

The source element enables multiple media sources to be specified for audio and video elements.

Empty

Each separate media source specified within a source element is a complete set of media data â€” for example, a complete video file, or a complete audio file).

src
The address of the media source.
type
The type of the media source (used for helping the UA determine, before fetching this media source, if it can play it).
media
The intended media type of the media source (used for helping the UA determine, before fetching this media source, if it is useful to the user).
pixelratio
For anamorphic media sources that do not self-describe their pixel ratio, specifies the ratio of the correct rendered width of each pixel to the actual height of each pixel in the image. If the pixelratio attribute is omitted or its value is not valid, source element represents a media source with a 1.0 pixel ratio.
