The table element

table

The table element represents a table; that is, data with more than one dimension.

In this order: optionally a caption element, followed by zero or more colgroup elements, optionally followed by a colgroup element, followed by one of the following:

  • a tfoot element, followed by either zero or more tbody elements or one or more tr
  • or, either zero or more tbody elements or one or more tr elements, optionally followed by a tfoot element
summary
A prose description of the structure of the table.
border
Indicates that its table element is not being used for layout purposes.

Although in previous versions of HTML a table element was only allowed to contain a tfoot element if that tfoot occurred before any tbody or tr elements, but not after, the table element is now also alternatively allowed to contain a tfoot after any tbody or tr elements.
