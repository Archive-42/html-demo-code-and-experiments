The time element

date and/or time

The time element represents a date and/or time.

Phrasing content
datetime
Specifies the date or time that the element represents.
datetime
Specifies the date that the element represents.
datetime
Specifies the date and time that the element represents.
pubdate

If the datetime attribute is not specified, then the date or time represented is given by the contents of the element, and the character data in those contents must conform to the time-datetime datatype.
