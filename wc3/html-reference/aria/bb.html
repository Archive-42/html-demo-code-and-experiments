bb – user-agent command # T

The bb element represents a user-agent command that the user can invoke.

Content model #

bb = (normal-character-data & common.elem.phrasing*) & bb.attrs

Attribute model #

bb.attrs = common.attrs & type?

Element-specific attributes #

type = "makeapp"
Specifies the action that its bb element represents. The value "makeapp" indicates that its action is to make a site accessible as a “standalone” application (as if it were not a Web site at all).

Assertions #

  • The interactive element “bb” must not appear as a descendant of the “a” element.
  • The interactive element “bb” must not appear as a descendant of the “button” element.
  • The interactive element “bb” must not appear as a descendant of the “bb” element.

Tag omission #

A bb element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing

Details #

When the bb element has no descendant element children and has no descendant text node children of non-zero length, the element represents a browser button with a UA-defined icon or text representing the action specified by the type attribute. Otherwise, the element represents its descendants.
