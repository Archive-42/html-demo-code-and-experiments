ⓘ nav – group of navigational links NEW # T

The nav element represents a section of a document that links to other documents or to parts within the document itself; that is, a section of navigation links.

Permitted contents #

flow content

Permitted attributes #

global attributes & common.attrs.aria.implicit.region or common.attrs.aria.landmark.navigation

ⓘ global attributes
Any attributes permitted globally.
region = #
(detailed attribute description to come)
navigation = #
(detailed attribute description to come)

Additional constraints and admonitions #

  • The sectioning element “nav” must not appear as a descendant of the “address” element.

Tag omission #

A nav element must have both a start tag and an end tag.

Permitted parent elements #

any element that can contain flow elements

DOM interface #

Uses HTMLElement.

Typical default display properties #

nav {
display: block; }
