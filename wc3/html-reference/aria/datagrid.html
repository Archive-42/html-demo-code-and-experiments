datagrid – datagrid # T

The datagrid element represents an interactive representation of tree, list, or tabular data.

Content model #

datagrid = (normal-character-data & common.elem.flow*) & datagrid.attrs

Attribute model #

datagrid.attrs = common.attrs & multiple? & disabled?

Element-specific attributes #

multiple = "multiple" | ""
Specifies that the user must be able to select any number of rows (zero or more).
disabled = "disabled" | ""
Instructs the UA to disable the datagrid, preventing the user from interacting with it.

Assertions #

  • The interactive element “datagrid” must not appear as a descendant of the “a” element.
  • The interactive element “datagrid” must not appear as a descendant of the “button” element.
  • The interactive element “datagrid” must not appear as a descendant of the “bb” element.

Tag omission #

A datagrid element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.flow
