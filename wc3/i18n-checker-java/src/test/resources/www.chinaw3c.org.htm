W3C发布HTML5和Canvas 2D的备选推荐标准更新

August 07, 2013 | 国际新闻,标准进展

W3C的HTML工作组于8月6日发布了两个备选推荐标准（Candidate Recommendations）的更新：

- HTML5. 这是万维网的核心语言 －可扩展标记语言的第5版。在这一版本中，增加了支持Web应用开发者的许多新特性，以及更符合开发者使用习惯的新元素，并重点关注定义清晰的、一致的准则，以确保Web应用和内容在不同用户代理（浏览器）中的互操作性。

- HTML Canvas 2D Context：定义了HTML canvas元素的2D上下文。2D Context为在Canvas上绘制和操纵图形提供了对象、方法、属性信息。

更多信息，请参阅W3C交互技术领域的HTML标准计划（HTML Activity）。 

备选推荐标准是W3C标准过程的一个环节，指W3C认为标准文本已经经过广泛的审查并满足了标准工作组定义的技术需求的技术文档，通过发布备选推荐标准向社会征求参考实现和使用经验，验证标准的可实现性。 更多信息，请参阅W3C标准制定流程。 

W3C Blog: W3C的工作重点（Highlights）

August 05, 2013 | 国际新闻,博客文章

8月1日，W3C首席运营官Jeff Jaffe博士在W3C官方博客撰文，介绍W3C在推进的重点工作。目前，W3C有50个工作组（Working Group）、15个兴趣小组（Interest Group）、132个社区组（Community Group）以及3个业务组（Business Group）。我们正在致力于不断增长的开发者关系及与其他标准组织的联络工作，以不断改进和增强Web平台。我们也在不断加强与数字出版、移动计算、汽车、电视与其他相关产业界的合作。要理解W3C的目标和思路并不容易。

W3C发布数据分类词汇表（DCAT）的工作草案最终征求意见稿

August 01, 2013 | 国际新闻,标准进展

W3C的政府互联数据工作组（Government Linked Data Working Group）8月1日发布了数据分类词汇表（Data Catalog Vocabulary，DCAT）的工作草案最终征求意见稿（Last Call Working Draft），该文档是一个用于政府在Web上发布数据时确定数据分类的RDF词汇表，定义了数据分类的大纲（schema）并给出了一组例子。欢迎您在2013年8月30日前提交您对该标准草案的意见和建议。

该工作组还同时发布了三分工作组备忘（Group Note）：

- 数据立方词汇表的用例和注意事项（Use Cases and Lessons for the Data Cube Vocabulary）：这是一个公开发布政府数据时使用数据立方词汇表的用例分析及使用中常见的问题汇总。

- 已注册的组织机构词汇表（Registered Organization Vocabulary）：这是一个描述在不同国家和地区注册组织机构的本体（ontology）的词汇表。

- 资产描述元数据大纲（Asset Description Metadata Schema, ADMS）： 它用来描述电子政务系统中基于高度可重用的元数据（如xml schema，或通用的数据模型）所定义的语义资产。

更多信息，请参阅电子政务标准计划（eGovernment Activity）。

W3C发布SCXML:State Machine Notation for Control Abstraction的标准草案最终征求意见稿

August 01, 2013 | 国际新闻,标准进展

W3C的语音浏览器工作组（Voice Browser Working Group）8月1日发布了状态图XML：状态机控制抽象符号（SCXML: State Machine Notation for Control Abstraction）的工作草案最终征求意见稿（Last Call Working Draft）。状态图可扩展标记语言（State Chart XML, SCXML) 是 W3C 组织制定的一种有限状态机的规范，它提供了一个在 CCXML 和 Harel State Tables 基础之上演化而来的状态机执行环境。其中，CCXML 是一种用来支持具有呼叫中心功能特点的语音应用程序基于事件驱动的状态机语言。CCXML 和 Harel State Tables 的结合搭建起了一套表达了富有规则的、很有想法的语义，并具有精密的构架逻辑。

我们欢迎您在2013年9月27日之前提交与该草案相关的意见和建议。

关于SCXML，可参阅开源中国社区的博客文章有限状态机规范SCXML。 

更多信息，请参阅W3C普适Web技术领域的语音浏览器标准计划（Voice Browser Activity）。

W3C的Web与电视兴趣小组制定新章程

July 30, 2013 | 国际新闻,标准进展,技术活动

W3C的Web与电视兴趣小组（Web and TV Interest Group）最近重新制定了小组章程。该兴趣小组主要推动针对未来电视产业的Web标准需求和研发，主要关注的Web标准包括加密媒体扩展（Encrypted Media Extensions, EME）、媒体源扩展（Media Source Extensions）、multi-track HTML视频和音频等。经过前期的工作，该兴趣小组启动了新的任务计划，并将小组重点确定为：

- 测试

- 录制和下载

- 终端能力

- 元数据

- 环绕立体声3D（Stereoscopic 3D）

- 事件序列文本/字幕

该兴趣小组还联系了一些主要的开发电视标准的行业组织、下一代支持网络的智能电视的软件开发和测试组织，包括OIPF, MMI以及HbbTV，同时保持对W3C在新兴市场领域进展的高度关注。

更多信息，请参阅Web与电视兴趣小组的新章程，并查阅工作组主页。

兴趣小组是W3C汇聚行业和产业声音，形成标准需求和标准意向的方式，面向行业及公众搭建交流平台。欢迎更多与智能电视设备、内容及播控厂商及组织关注该兴趣小组的工作。

W3C发布IndieUI: Events 1.0的标准工作草案

July 30, 2013 | 国际新闻,标准进展

W3C的IndieUI工作组7月30日发布了IndieUI: Events 1.0的工作草案更新（updated Working Draft）。IndieUI定义了一组与用户输入方式无关的事件，将依赖于用户输入方式的事件翻译为不依赖于用户输入方式的“高层”简单事件，基于这些简单事件编程可以简化Web应用的设计。例如，如果用户希望向下滚动页面内容，他可以用手指在触摸屏上向下滑动，也可以用鼠标上的滚动球向下滚动，或者使用滚轮，甚至在有些应用中通过语音命令来操纵。在IndieUI中，我们可以定义一个简单的Scroll Down事件，并将上述可能的操作关联到这一简单事件上。这在编写需要面对各种输入设备的Web应用时，将极大简化Web应用的开发。

我们欢迎您在2013年9月13日前提交您对该标准草案的意见和建议。

更多信息，请参阅IndieUI的概述，及相关的电子邮件。

更多信息，请参阅Web可访问性标准计划（Web Accessibility Initiative）。

July 30, 2013 | 国际新闻,标准进展

W3C的互联数据平台（Linked Data Platform, LDP）工作组发布了互联数据平台（Linked Data Platform）1.0版的工作草案最终征求意见稿（Last Call Working Draft）。该文档描述了一组支持数据读写的互联数据体系架构（read-write linked data architecture）的最佳实践和简单的实现方式，它基于HTTP协议访问Web资源，并使用RDF数据模型描述资源的状态。我们欢迎您于2013年9月2日前提交您的意见和建议。
