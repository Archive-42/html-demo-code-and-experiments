This is the filled out registration template for the URI scheme described in:

  http://dev.w3.org/2006/waf/widgets-uri/

URI scheme name.
	widget

Status.
	permanent
	(Provisional registration is also acceptable.)

URI scheme syntax.
	Generic URI.

URI scheme semantics.
	Locating resources at runtime inside of a widget package.

Encoding considerations.
	n/a

Applications/protocols that use this URI scheme name.
	This scheme is used in user-agents that implement W3C Widgets [WIDGETS]

Interoperability considerations.
	n/a

Security considerations.
	Since such URIs cannot be used to address content outside of a widget package,
	they introduce no specific further security considerations.
