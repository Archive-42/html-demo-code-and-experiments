What does `<ruby> HTML Tag` do?  
The &lt;ruby&gt; element is used pair characters of certain Asian languages with pronunciation information. The &lt;ruby&gt; element is used in conjunction with the &lt;rp&gt; and &lt;rt&gt; elements.

Display  
inline

Code Example
------------

    <ruby><rb>ruby base<rt>annotation</ruby>

ruby baseannotation

<span class="underline"></span>

Hints and Annotations For Asian Languages
-----------------------------------------

Ruby is used to display pronunciation hints in some Asian languages. It can also display small annotations that are relevant to the main content. Ruby is used in some Asian languages, including Japanese and Taiwanese.

The `rb` and `rt` elements are used to identify the portion of the `ruby` element that is the base and the portion that is the ruby text or annotation.
