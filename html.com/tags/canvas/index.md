What does `<canvas> HTML Tag` do?  
The &lt;canvas&gt; element creates a rectangular pane of arbitrary size which can be used for drawing graphics, manipulating photos, and creating animations with JavaScript.

Drawing with JavaScript
-----------------------

The `<canvas>` element provides a panel of arbitrary size which can be used to display images and other content via [JavaScript](https://html.com/javascript/).
