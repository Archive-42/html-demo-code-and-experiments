What does `<small> HTML Tag` do?  
The &lt;small&gt; element identifies text to display one size smaller than the surrounding text. In HTML5 the element is intended to be used to identify items of secondary importance such as copyright notices, side comments, and legal notices.

<span id="A_CSS_Alternative">A CSS Alternative</span>
-----------------------------------------------------

The `<small>` element makes text one size smaller than the default text size on the page. It’s offered as an alternative to CSS styling. `<small>` is a rare example of a HTML element that can be nested. In other words, you can use it several times to step down the font size, as in our example.

### <span id="Increasing_Text_Size">Increasing Text Size</span>

To return text to its default size, use `</small>`. To increase its size, use [`<big>`](https://html.com/tags/big/).
