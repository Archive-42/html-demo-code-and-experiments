What does `<s> HTML Tag` do?  
The &lt;s&gt; element is used to identify text that is no longer accurate or relevant. It is similar to, but semantically distinct from, the &lt;del&gt; element which is used to identify document edits. By default, browsers render the contents of an &lt;s&gt; element with a strikethrough.

## <span id="Crossing_Out">Crossing Out</span>

`<s>` is used to strike out text. The content of the `<s>` tag is displayed with a line through it. `<s>` was removed from the HTML 4 specification, but has been reintroduced in HTML 5 as a way to mark something that is no longer correct.

## <span id="Correct_Usage">Correct Usage</span>

In previous iterations of HTML, `<strike>`, that was used for the same purpose as `<strike>`. However, `<strike>` is now deprecated in HTML 5. Additionally, note that the [`<del>`](https://html.com/tags/del/) element should be used for text that has been _deleted_, rather than incorrect, since `<del>` provides the option to add a date stamp.
