What does `Creating A Button With The HTML Button Element: Here's How` do?  
The &lt;button&gt; element is used to create an HTML button. Any text appearing between the opening and closing tags will appear as text on the button. No action takes place by default when a button is clicked. Actions must be added to buttons using JavaScript or by associating the button with a form.

------------
