What does `<xmp> HTML Tag` do?  
The &lt;xmp&gt; element was used to surround HTML example text that should be rendered without interpreting any HTML elements between the opening and closing &lt;xmp&gt; tags. The element was deprecated in HTML 3.2 and is now obsolete.

Display  
inline

Usage  
textual

<span class="underline"></span>

`<xmp>` was designed to render content exactly as typed, without compressing spaces. This was intended as a way to display code. However, this element was never fully supported and is now obsolete.

[Read more about `<xmp>`, and the reasons for its deprecation](http://www.the-pope.com/listin.html).
