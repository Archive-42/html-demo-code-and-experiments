What does `<font> HTML Tag` do?  
The &lt;font&gt; element was used to specify typographical display styles. It has been deprecated and fonts should be styled with CSS instead.

How to style fonts
------------------

The `<font>` element, along with most other style and design oriented HTML features, has been [deprecated in HTML5](https://html.com/html5/). The correct way to control fonts and other styling is through [CSS](https://html.com/css/). For more information, see our [tutorial on web typography](https://html.com/fonts/).
