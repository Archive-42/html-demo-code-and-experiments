What does `How To Use  In Your HTML` do?  
The &lt;noscript&gt; element contains HTML content that will be rendered if a user viewing the webpage does so using a browser that does not support scripts or has disabled scripts.

Display  
inline

<span class="underline"></span>

The `noscript` element defines a message that is displayed if the browser does not support a defined [`script`](https://html.com/tags/script/). For example, we could use the following example to alert the user that JavaScript is switched off:

    <script src="myscript.js" type="text/javascript"></script> <noscript>Please enable JavaScript in your browser.</noscript>
