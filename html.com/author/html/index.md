    [&lt;track&gt; HTML Tag](https://html.com/tags/track/)
    ------------------------------------------------------

    The &lt;track&gt; element is used as a child of an &lt;audio&gt; or &lt;video&gt; element and adds a time-based data source to the parent media element. For example, the &lt;track&gt; element can be used to add timed subtitles to a video and closed captions to audio content.

    [&lt;time&gt; HTML Tag](https://html.com/tags/time/)
    ----------------------------------------------------

    Defines a datetime. The content of a time element is typically a human-readable date and time, and a machine-readable version of the same time is placed in the datetime attribute.

    [&lt;tfoot&gt; HTML Tag](https://html.com/tags/tfoot/)
    ------------------------------------------------------

    The &lt;tfoot&gt; element identifies one or more &lt;tr&gt; elements as containing summary contents of a table's columns. The &lt;tfoot&gt; element must be the direct descendant of a &lt;table&gt; element. In HTML5, &lt;tfoot&gt; can be placed either before or after &lt;tbody&gt; and &lt;tr&gt; elements, but must appear after any &lt;caption&gt;, &lt;colgroup&gt;, and &lt;thead&gt; elements.

-   <img src="https://html.com/wp-content/uploads/Code.jpg" class="attachment-oncanvas-large-thumbnail default-featured-img sp-no-webp" sizes="(max-width: 340px) 100vw, 340px" srcset="

    [&lt;tbody&gt; HTML Tag](https://html.com/tags/tbody/)
    ------------------------------------------------------

    The &lt;tbody&gt; element must be a direct descendant of a &lt;table&gt; element and is used to identify &lt;tr&gt; elements that comprise the body of the table. The &lt;tbody&gt; element should always come after a &lt;thead&gt; element and may come before or after a &lt;tfoot&gt; element.

    ----------------------------------------------------------

    The &lt;summary&gt; element is used as a child of &lt;details&gt; element to provide a summary of the contents of the &lt;details&gt; element. At this time, &lt;summary&gt; is not well supported across browsers.

    [&lt;dialog&gt; HTML Tag](https://html.com/tags/dialog/)
    --------------------------------------------------------

    The &lt;dialog&gt; element is used to create a dialog box such as a popup or modal window that is rendered within the active browser window. Defining the behavior of a &lt;dialog&gt; element typically requires the use of JavaScript.

    [&lt;details&gt; HTML Tag](https://html.com/tags/details/)
    ----------------------------------------------------------

    The &lt;details&gt; element is used to pair a &lt;summary&gt; statement with additional related details. The &lt;summary&gt; is displayed, and a user can view or hide additional details by clicking on the summary.
