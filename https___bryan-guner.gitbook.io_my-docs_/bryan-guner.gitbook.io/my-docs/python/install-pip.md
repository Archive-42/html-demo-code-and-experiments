If you prefer a command-line method, you can use </span>`wget`<span data-offset-key="009561aa8ee342a79da7daa8cdf26da4:4"> to download the file to your current directory:</span></span>

1

<span data-key="6e4e50ef94c84718b4a03ec1dcd7f44f"><span data-offset-key="6e4e50ef94c84718b4a03ec1dcd7f44f:0">$ wget https://www.python.org/ftp/python/3.8.4/Python-3.8.4.tgz</span></span>

Copied!

<span data-key="fbe7286519b44de2bc72c85d6f901856"><span data-offset-key="fbe7286519b44de2bc72c85d6f901856:0">When the tarball finishes downloading, there are a few things you’ll need to do to prepare your system for building Python.</span></span>

<span data-key="bc975556922247cca56b3bb6d5b1d621">**Step 2: Prepare Your System**</span>

<span data-key="cd214d97fd22428e8117d6f73529df98"><span data-offset-key="cd214d97fd22428e8117d6f73529df98:0">There are a few distro-specific steps involved in building Python from scratch. The goal of each step is the same on all distros, but you might need to translate to your distribution if it doesn’t use </span>`apt-get`<span data-offset-key="cd214d97fd22428e8117d6f73529df98:2">:</span></span>

1.  <span class="r-1awozwy r-1nf4jbm r-6koalj r-18u37iz r-gg6oyi r-ubezar r-16dba41 r-135wba7 r-1m04atk r-1pyaxff" contenteditable="false" style="height: 24px">1.</span>
    <span data-key="a02fe262bd3a4acaaeadd45862438176"><span data-offset-key="a02fe262bd3a4acaaeadd45862438176:0">First, update your package manager and upgrade your packages:</span></span>

    1

    <span data-key="8f1a9636aa694b0c98e06f301d9f779b"><span data-offset-key="8f1a9636aa694b0c98e06f301d9f779b:0">$ sudo apt-get update</span></span>

    2

    <span data-key="bdd37383f5f94965816edf066c0f1343"><span data-offset-key="bdd37383f5f94965816edf066c0f1343:0">$ sudo apt-get upgrade</span></span>

    Copied!

2.  <span class="r-1awozwy r-1nf4jbm r-6koalj r-18u37iz r-gg6oyi r-ubezar r-16dba41 r-135wba7 r-1m04atk r-1pyaxff" contenteditable="false" style="height: 24px">2.</span>
    <span data-key="102db9e1537f46a8a4d1d735b1308373"><span data-offset-key="102db9e1537f46a8a4d1d735b1308373:0">Next, make sure you have all of the build requirements installed:</span></span>

    1

    <span data-key="2faedd331ae34b2da4ddc89ff731da33"><span data-offset-key="2faedd331ae34b2da4ddc89ff731da33:0">\# For apt-based systems (like Debian, Ubuntu, and Mint)</span></span>

    2

    <span data-key="e944ca7d4b8647779ab2f13fb88032a8"><span data-offset-key="e944ca7d4b8647779ab2f13fb88032a8:0">$ sudo apt-get install -y make build-essential libssl-dev zlib1g-dev \\</span></span>

    3

    <span data-key="4a8dce2fded04d879e0eb7acf8e07f63"><span data-offset-key="4a8dce2fded04d879e0eb7acf8e07f63:0"> libbz2-dev libreadline-dev libsqlite3-dev wget curl llvm \\</span></span>

    4

    <span data-key="2a7cee7f5478446a9e14c48a9e989392"><span data-offset-key="2a7cee7f5478446a9e14c48a9e989392:0"> libncurses5-dev libncursesw5-dev xz-utils tk-dev</span></span>

    5

    <span data-key="d9fdd014e7b847c8995e5f5af0d360a3"><span data-offset-key="d9fdd014e7b847c8995e5f5af0d360a3:0"><span data-slate-zero-width="n">​</span></span></span>

    6

    <span data-key="c5913ea458804373993da17f6e019d98"><span data-offset-key="c5913ea458804373993da17f6e019d98:0">\# For yum-based systems (like CentOS)</span></span>

    7

    <span data-key="4bf404d59d7349eb8f2e646be60b29e2"><span data-offset-key="4bf404d59d7349eb8f2e646be60b29e2:0">$ sudo yum -y groupinstall "Development Tools"</span></span>

    8

    <span data-key="fa4605450e4f4220a1f5218b6d7fdafa"><span data-offset-key="fa4605450e4f4220a1f5218b6d7fdafa:0">$ sudo yum -y install gcc openssl-devel bzip2-devel libffi-devel</span></span>

    Copied!

    <span data-key="14185b45bb3c45a7abd2c2ac7d8834a8"><span data-offset-key="14185b45bb3c45a7abd2c2ac7d8834a8:0">It’s fine if you already have some of the requirements installed on your system. You can execute the above commands and any existing packages will not be overwritten.</span></span>

<span data-key="2757f020b4f74008bbe3387452b827a2"><span data-offset-key="2757f020b4f74008bbe3387452b827a2:0">Now that your system is ready to go, it’s time to start building Python!</span></span>

<span data-key="594e580f00f84b4e862d8bbc45268cd8">**Step 3: Build Python**</span>

1.  <span class="r-1awozwy r-1nf4jbm r-6koalj r-18u37iz r-gg6oyi r-ubezar r-16dba41 r-135wba7 r-1m04atk r-1pyaxff" contenteditable="false" style="height: 24px">1.</span>
    <span data-key="7c90d51320204c10b01e9a1dde84d356"><span data-offset-key="7c90d51320204c10b01e9a1dde84d356:0">Once you have the prerequisites and the TAR file, you can unpack the source into a directory. Note that the following command will create a new directory called </span>`Python-3.8.3`<span data-offset-key="7c90d51320204c10b01e9a1dde84d356:2"> under the one you’re in:</span></span>

    1

    <span data-key="c6db04f63bac4809a1500c906d905bb3"><span data-offset-key="c6db04f63bac4809a1500c906d905bb3:0">$ tar xvf Python-3.8.4.tgz</span></span>

    2

    <span data-key="fd207256f2d94c9fa70d7648d6611259"><span data-offset-key="fd207256f2d94c9fa70d7648d6611259:0">$ cd Python-3.8.4</span></span>

    Copied!

2.  <span class="r-1awozwy r-1nf4jbm r-6koalj r-18u37iz r-gg6oyi r-ubezar r-16dba41 r-135wba7 r-1m04atk r-1pyaxff" contenteditable="false" style="height: 24px">2.</span>
    <span data-key="f3cc64b1be8f45168d052f24dd19b0f3"><span data-offset-key="f3cc64b1be8f45168d052f24dd19b0f3:0">Now you need to run the </span>`./configure`<span data-offset-key="f3cc64b1be8f45168d052f24dd19b0f3:2"> tool to prepare the build:</span></span>

    1

    <span data-key="5d7f7350578a443a887ea85ac36d2633"><span data-offset-key="5d7f7350578a443a887ea85ac36d2633:0">$ ./configure --enable-optimizations --with-ensurepip=install</span></span>

    Copied!

    <span data-key="e5322bc357274497a0ae77e07e549eac"><span data-offset-key="e5322bc357274497a0ae77e07e549eac:0">The </span>`enable-optimizations`<span data-offset-key="e5322bc357274497a0ae77e07e549eac:2"> flag will enable some optimizations within Python to make it run about 10 percent faster. Doing this may add twenty or thirty minutes to the compilation time. The </span>`with-ensurepip=install`<span data-offset-key="e5322bc357274497a0ae77e07e549eac:4"> flag will install </span>`pip`<span data-offset-key="e5322bc357274497a0ae77e07e549eac:6"> bundled with this installation.</span></span>

3.  <span class="r-1awozwy r-1nf4jbm r-6koalj r-18u37iz r-gg6oyi r-ubezar r-16dba41 r-135wba7 r-1m04atk r-1pyaxff" contenteditable="false" style="height: 24px">3.</span>
    <span data-key="61a8f85bdabd41e2a1fee33005ae0f2d"><span data-offset-key="61a8f85bdabd41e2a1fee33005ae0f2d:0">Next, you build Python using </span>`make`<span data-offset-key="61a8f85bdabd41e2a1fee33005ae0f2d:2">. The </span>`-j`<span data-offset-key="61a8f85bdabd41e2a1fee33005ae0f2d:4"> option simply tells </span>`make`<span data-offset-key="61a8f85bdabd41e2a1fee33005ae0f2d:6"> to split the building into parallel steps to speed up the compilation. Even with the parallel builds, this step can take several minutes:</span></span>

    1

    <span data-key="bea0d195463345fc8215ef38506dbb34"><span data-offset-key="bea0d195463345fc8215ef38506dbb34:0">$ make -j 8</span></span>

    Copied!

4.  <span class="r-1awozwy r-1nf4jbm r-6koalj r-18u37iz r-gg6oyi r-ubezar r-16dba41 r-135wba7 r-1m04atk r-1pyaxff" contenteditable="false" style="height: 24px">4.</span>
    <span data-key="b21870751b314d289dbbc39c67d2979f"><span data-offset-key="b21870751b314d289dbbc39c67d2979f:0">Finally, you’ll want to install your new version of Python. You’ll use the </span>`altinstall`<span data-offset-key="b21870751b314d289dbbc39c67d2979f:2"> target here to avoid overwriting the system Python. Since you’re installing into </span>`/usr/bin`<span data-offset-key="b21870751b314d289dbbc39c67d2979f:4">, you’ll need to run as root:</span></span>

    1

    <span data-key="bebf855129e042a0b3043022962fdc63"><span data-offset-key="bebf855129e042a0b3043022962fdc63:0">$ sudo make altinstall</span></span>

    Copied!

<span data-key="184ac6b2d43343589a7068a225da12d6"><span data-offset-key="184ac6b2d43343589a7068a225da12d6:0">It might take a while to finish installation. Once it’s done, you can verify that Python is set up correctly.</span></span>

<span data-key="6c80556e9b40444e8eeb17ea6bd5c869">**Step 4: Verify Your Installation**</span>

<span data-key="8ebe8e64a3684f21a202ba6b3e8f9df2"><span data-offset-key="8ebe8e64a3684f21a202ba6b3e8f9df2:0">Test that the </span>`python3.8 --version`<span data-offset-key="8ebe8e64a3684f21a202ba6b3e8f9df2:2"> command returns the latest version:</span></span>

1

<span data-key="5d04c6431b7e4947802e0caf326b68ab"><span data-offset-key="5d04c6431b7e4947802e0caf326b68ab:0">$ python3.8 --version</span></span>

2

<span data-key="e1462bcd38884afd94a1db642f78b8da"><span data-offset-key="e1462bcd38884afd94a1db642f78b8da:0">Python 3.8.4</span></span>

Copied!

<span data-key="30288fc605e34b3496a30b6e3f26240e"><span data-offset-key="30288fc605e34b3496a30b6e3f26240e:0">If you see </span>`Python 3.8.4`<span data-offset-key="30288fc605e34b3496a30b6e3f26240e:2">, then you’re all set!</span></span>

<span data-key="2016e85fdc9c488cbe54079a12377054"><span data-offset-key="2016e85fdc9c488cbe54079a12377054:0">If you have some extra time on your hands, you can also run the </span></span><a href="https://devguide.python.org/runtests/" class="css-4rbku5 css-1dbjc4n r-1loqt21 r-1471scf r-1otgn73 r-1i6wzkk r-lrvibr"><span data-key="01edb69a337a446b972184cef0f11d10" data-rnw-int-class="nearest_260-8535_262-8536-240__"><span data-key="b701d68125b74f0fae20fc304579516d"><span data-offset-key="b701d68125b74f0fae20fc304579516d:0">test suite</span></span></span></a><span data-key="bc68a6253a974357b823405f6dfdd0c0"><span data-offset-key="bc68a6253a974357b823405f6dfdd0c0:0"> to make sure everything is working properly on your system.</span></span>

<span data-key="c12fe16f23ed4698bfa9b8d50b99d43d"><span data-offset-key="c12fe16f23ed4698bfa9b8d50b99d43d:0">To run the test suite, type the following command:</span></span>

If all the tests pass, then you can be confident that your brand-new Python build is working as expected!</span></span>
